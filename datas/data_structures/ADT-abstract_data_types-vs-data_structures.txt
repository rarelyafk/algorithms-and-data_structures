An ADT (Abstract Data Type) is an abstraction
of a DS (Data Structure) which provides only
the interface to which a data structure
must adhere to.
(how they behave and what methods)
NOT the specific details/implementation of such.
(how implemented)

      ADT         -VS-      DS
mode of transport       bicycle, car, train
list                    dynamic array
                        linked list
queue                   linked list based queue
                        array based queue
                        stack based queue
map                     tree map
                        hash map/table
